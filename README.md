About this web app: 
In this project, I built a Node.js web application that displays events happening in New York City. The application connects to a PostgreSQL database hosted on Render to store and retrieve event information.

I created the database tables, inserted sample event data, and connected the backend server to the database using environment variables. The application allows users to view a list of events and see details such as the event name, date, time, venue, genre, ticket price, image, and description.

To test the project, I ran the server locally using npm and confirmed that the data loads from the PostgreSQL database. The database is deployed and running on Render, and the project is managed using GitHub.
